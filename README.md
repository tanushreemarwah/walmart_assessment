# Walmart Homework Assignment

Create a simple application to list Users and show User details page using Fake JSON APIs. 

## This application is written in : 

* HTML5
* CSS
* (Vanilla) JavaScript
* Bootstrap 5
* Typicode - Fake REST APIs

# Functionality

This application supports the following functionality: 

1. Display a list of Users by Name
2. Hover on nay list item to show User's email and username
3. Click on the list item to navigate to the User Details Page
4. User Details Page has three sections - Profile, Posts and Albums
5. Posts section is collapsed by default, opens to show all posts of the user and comments on each post
7. Albums section is collapsed by default, opens to show a list of user's albums
8. Click on any album title to open a modal showing a photo carousel
